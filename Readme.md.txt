🌀 FinGenie AI
Intelligent Digital Banking & Wealth Management Platform
 
A full-stack AI-powered digital banking platform built with **Spring Boot** and **ReactJS**.
 
---
🚀 Tech Stack
 
| Layer | Technology |
|-------|-----------|
| Frontend | ReactJS, Redux, Axios |
| Backend | Java 17, Spring Boot 3.x |
| Security | JWT, BCrypt, OTP |
| Database | MySQL, Spring Data JPA |
| Tools | Maven, Lombok, Postman |
 

 
✨ Features
 
🔐 JWT + OTP Two Factor Authentication
🏦 Deposit, Withdraw, Transfer
🚨 AI Fraud Detection
📈 Investment Advisory
💰 Loan Management & EMI Calculator
🤖 AI Chat Assistant
👨‍💼 Admin Analytics Dashboard
📱 QR Code Payments
 
 🛠️ Setup & Run
 
Backend
bash
cd fingenie-backend
mvn spring-boot:run


Frontend
cd fingenie-drontend
npm install
npm start
